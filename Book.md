## **Objectifs principaux :**

- Animation win
- Créer un solitaire avec un timer
- Créer une base de données

- **Système de matchmaking :**
  - File d'attente (timer, info) (**socket 3/4 pts**)

- **Profil :**
  - Possibilité de lié sont compte (récupérer etc...) (**2 pts**)
